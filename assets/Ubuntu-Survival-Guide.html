A Survival Guide

💪 Super Powers

Run a command as Super User
sudo COMMAND

Elevate to root
sudo -i

Turn a user into a Super User
sudo usermod -aG sudo USER

🔧 System Maintenance

Show Kernel version (-a for all)
uname -r

Show Ubuntu version (-a for all)
lsb_release -r

Keep the system up-to-date
sudo apt update && sudo apt dist-upgrade

Keep the system clean (by deleting useless files)
sudo apt autoremove && sudo apt autoclean

Upgrade to a newer release (system must be up-to-date first)
sudo do-release-upgrade -d

List all running processes
ps aux

Kill a process
kill -9 PROCESS_ID

See which programs are listening on which ports
netstat -peanut

See who is listening on a specific port
lsof -i :PORT

Run a command as a service (console won't hang)
COMMAND &

Find a previous command in the History
history | grep COMMAND

Run a previous command found in the History
!COMMAND_ID

Check out System Logs (other logs are in the same folder)
cat /var/log/syslog (or less or tail or nano or...)

📁 Files and Folders

List files with details
ls -l

List all files including hidden files
ls -la

List files ordered by last modified
ls -lt

Clone a directory (recursive, same timestamps and permissions)
cp -a FOLDER TARGET

Find a file from current directory
sudo find . -name 'FILE'

Read the last 1000 rows of a file and listen for modifications
tail -1000f FILE

Read a file interactively (h for help)
less FILE

Get the size of a directory
du -sh FOLDER

Check partitions status (is your disk full?)
df -h

Relax permissions on a file or a folder:
chmod 777 TARGET

Set ownership (user/group) of a file or a folder
chown USER:GROUP TARGET

Open a file in a privileged terminal editor
sudo nano FILE

Find occurrences of a text in a file or in many files
grep TEXT TARGET

👥 Users and Groups

List all groups
groups

List groups a user is a member of
groups USER

List users which are members of a group
getent group GROUP

Create a new user
sudo adduser USER

Add an existing user to a group
sudo usermod -aG GROUP USER

Change a user's password
passwd USER

Impersonate a user (or return to your user from root)
su USER

Move to user home (AKA ~, AKA /home/USER)
cd

Delete a user
sudo userdel USER

🎨 Graphical User Interface

Open a Terminal
CTRL + ALT + T

Open a privileged file explorer (for current folder use .)
sudo nautilus PATH

Open a file in a privileged GUI text editor
sudo gedit FILE
